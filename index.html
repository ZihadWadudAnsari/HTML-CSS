Bootcamp 2020: "CryptoGr4ph2019" application

This my first programming project, concluding the 2-week Programming Bootcamp at City (2020). This Processing project demonstrates a comprehensive understanding of the use of arrays, if statements, for loops, map function and many types of variables. I created a data representation of four different types of cryptocurrency (BTC, ETH, LTC, XRP) with their prices from 01/01/2019 to 01/12/2019 (the first date of each month). I had to ensure that the Y values would be appropriate to the particular cryptocurrency values. The data is presented as a Scatter plot graph.
